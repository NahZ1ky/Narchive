4.
   Rî = Aî + Bî
      = 40 * cos(15°) + 80 * cos(120°)
      = 38.66 - 39.70
      = -1.04
      
   Rĵ = Aĵ + Bĵ
      = 40 * sin(15°) + 80 * sin(120°)
      = 10.35 + 69.28
      = 79.63

    R = √(Rî² + Rĵ²)
      = √(-1.04² + 79.63²)
      = √(1.08 + 634)
      = 79.6 m
   Magnitude = *79.6* m

   θ = tan⁻¹(Rĵ / Rî)
     = tan⁻¹(634 / 1.08)
     = tan⁻¹(587)
     = 89.9°
   Direction = *89.9* °
